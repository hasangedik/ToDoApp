# ToDoApp
This is a simple SOA based Asp.Net MVC to-do list application.

Architectural Info
-Service Oriented
-Basit HMAC Authentication on WebApi
-Logs on MongoDb
-Entity Framework (Code First)
-Asp.Net MVC WebSite

Used Frameworks
-Castle Windsor
-Log4net
-ExpressMapper
-Hangfire
-RestSharp
etc.
